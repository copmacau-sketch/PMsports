Create Safe Code

To enhance the security of your account, Safe Code is required during login. Please create your preferred Safe Code.
  • 1. Safe Code must be between 8 - 16 alphanumeric characters with at least 1 upper (A - Z) and lowercase letters (a - z) and at least 1 numeric character (0 - 9).
  • 2. Safe Code must contain at least 3 different alphanumeric characters.
  • 3. Safe Code must NOT contain any spaces.

Your Login ID must be between 6 to 12 alphanumeric (A-Z & 0-9) characters

NOTE

Please remember the Safe Code you have created. You will need to enter this on your next login. Please take note of the Safe Code entered before submitting.
